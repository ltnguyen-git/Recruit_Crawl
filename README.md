# Recruit_Data_Crawl

This is a example data crawler project that scrapes data from websites

## Features
- Crawl data from websites using Python
- Crawl from:

  → https://glints.com/vn/
  
  → https://careerbuilder.vn/
  
  → https://www.topcv.vn/
  
  → https://itviec.com/


## Requirements
- Python 3.x
- Beautiful Soup 4
- Requests
- Selenium
